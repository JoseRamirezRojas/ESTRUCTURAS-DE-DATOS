Pedroogle 1.0

Descripción: Nuestro programa Pedroogle tiene un menú con opciones del numero 1 al numero 3, 
que permiten hacer una búsqueda de términos entre una lista de archivos; agregar un archivo ".txt"; y la opcion de salir del programa, respectivamente.


Cabe señalar que Pedroogle sólo acepta archivos ".txt", de intentarse agregar otro tipo de archivo el programa no lo permitirá.

Si se intenta hacer una búsqueda con una lista vacía se pedirá inmediatamente ingresar una búsqueda de algún archivo.

Tanto para realizar una búsqueda como para agregar un archivo, es necesario una vez escritos los términos de la búsqueda, o bien, la ruta del archivo,
presionar Ctrl+D para que lo escrito sea leído por el programa.

El programa muestra los archivos de mayor a menor coincidencia. Una vez mostrados los resultados, Pedroogle te redirecciona al menú principal del programa.
